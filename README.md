# VID3 Video Player

Video player for HLS streaming on mobile and desktop. 

## Install instructions

### Requirements
* nodejs v16.13.2
* mpn v8.1.2

### Commands 
* Fetch repo
```
git clone https://github.com/harmony-one/vid3-video-player.git

```
* Install dependencies
```
yarn
```
* Development
```
yarn start
```
* Build
```
yarn build
```
The build artificats will be in /build folder.
